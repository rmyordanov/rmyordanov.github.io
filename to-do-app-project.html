To Do App

I have always wanted to create such an app. Finally, I did it. Except for the obvious functions for creating completing and deleting tasks, you can also edit them, and add notes to the task and track their. You can also drag and drop a task into different categories. There is a lot of things I want and will improve in the future. This is just the first version of the app.

Project Purpose and Goal

Purpose: The purpose of this app is to help users increase their productivity by providing them with a user-friendly platform to organize their tasks, take notes, and track the time they spend on each task.

Features:

  • 1. To-Do List: Users can create and manage their tasks by adding, editing, and deleting items from their to-do list.
  • 2. Note-Taking: Users can take notes related to their tasks, which will be saved and linked to the corresponding task for easy access.
  • 3. Drag and Drop: Users can easily reorder their tasks by using the drag and drop feature.
  • 4. Time Tracking: Users can track the time they spend on each task, which will be recorded and displayed on the task details page. This feature will help users to understand how they are spending their time and make adjustments to their workflow if necessary.

Web Stack and Explanation

To-Do App is built using a web stack of React for building the UI, JavaScript for creating the app's logic and functionality, and Firebase for handling the back-end services such as real-time database, user authentication. And it is hosted on GitHub Pages.
